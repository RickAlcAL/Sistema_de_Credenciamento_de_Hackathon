# Sistema_de_Credenciamento_de_Hackathon
Sistema robusto em Java voltado à triagem e processamento automático de cadastros para Hackathons. A solução valida dados inconsistentes a partir de arquivos brutos textuais, separa os perfis qualificados de Desenvolvedores e Designers seguindo regras de negócio rígidas (como faixa etária) e exporta relatórios organizados de aprovados e pendências.
